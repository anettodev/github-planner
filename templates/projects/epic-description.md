# Epic Description Template

Use for: GitHub Project (v2) description body.

```markdown
## {Epic Title}

{1-3 sentence description of the epic's goal and scope}

**Source**: `{path/to/planning-document.md}`
**Created**: {YYYY-MM-DD}
**Total issues**: {N}
**Phases**: {comma-separated phase names}

### Phases

| Phase | Issues | Description |
|---|---|---|
| {Phase 1 name} | {N} | {brief description} |
| {Phase 2 name} | {N} | {brief description} |

### Links
- Source document: `{path}`
- Repository: {owner/repo}
```
